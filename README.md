Théorie de l'algèbre de Clifford $Cl(0,3)$
Introduction
Exposition des propriétés fondamentales de l'algèbre $Cl(0,3)$.

Base de l'espace
La base est composée de trois vecteurs ${e_1, e_2, e_3}$ respectant les relations suivantes :

• $e_i^2 = -1$

• $e_i e_j = -e_j e_i$ pour $i \neq j$

Structure de l'algèbre
Cette algèbre est de dimension 8.
